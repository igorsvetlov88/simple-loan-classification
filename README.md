Репозиторий берет данные для классификации из этого датасета:
https://www.kaggle.com/datasets/sujithmandala/simple-loan-classification-dataset


Описания столбцов:

возраст (int): Возраст человека.
пол (строка): Пол человека, либо "мужской", либо "женский".
род занятий (строка): Род занятий человека.
образование_уровень (строка): Наивысший уровень образования, достигнутый данным лицом, такой как "Средняя школа", "Младший научный сотрудник", "Бакалавр", "Магистр" или "докторантура".
семейный_статус (строка): Семейное положение физического лица, либо "Холост", либо "Женат".
доход (int): Годовой доход физического лица в долларах.
credit_score (int): Кредитный рейтинг физического лица, варьирующийся от 300 до 850.
loan_status (строка): Целевая переменная, указывающая, была ли заявка на получение кредита "одобрена" или "Отклонена".
Этот набор данных содержит различные демографические и финансовые характеристики, которые могут быть использованы для построения классификационной модели для прогнозирования одобрения или отказа в выдаче кредита. Столбец loan_status служит целевой переменной для задачи классификации.
