Репозиторий берет данные для классификации из этого датасета:
https://www.kaggle.com/datasets/parthpatel2130/realistic-loan-approval-dataset-us-and-canada/data


Описания столбцов:

customer_id
Тип: object
уникальный идентификатор клиента
не используется в обучении модели, так как не несёт предсказательной информации


age
Тип: int
возраст клиента 


occupation_status
Тип: object
статус занятости клиента (например, employed, self-employed и т.д.)


years_employed
Тип: float
количество лет трудовой деятельности клиента


annual_income
Тип: int
годовой доход клиента


credit_score
Тип: int
кредитный рейтинг клиента, отражающий его платёжную дисциплину


credit_history_years
Тип: float
длительность кредитной истории (в годах)


defaults_on_file
Тип: int
наличие дефолтов в кредитной истории (0 — нет, 1 — есть)


delinquencies_last_2yrs
Тип: int
количество просрочек платежей за последние 2 года


derogatory_marks
Тип: int
количество негативных отметок в кредитной истории


savings_assets
Тип: int
объём сбережений и ликвидных активов клиента

current_debt
Тип: int
текущая долговая нагрузка клиента


product_type
Тип: object
тип кредитного продукта (например, personal, mortgage и т.д.)


loan_intent
Тип: object
цель кредита (образование, бизнес, жильё и др.)


loan_amount
Тип: int
сумма запрашиваемого кредита


interest_rate
Тип: float
процентная ставка по кредиту



debt_to_income_ratio
Тип: float
отношение общей задолженности клиента к его доходу


loan_to_income_ratio
Тип: float
отношение суммы кредита к годовому доходу клиента


payment_to_income_ratio
Тип: float
доля ежемесячного платежа по кредиту от дохода клиента


Целевая переменная
loan_status
Тип: int

Значения:

1 — кредит одобрен

0 — кредит отклонён

Описание: результат рассмотрения кредитной заявки
Этот набор данных содержит различные демографические и финансовые характеристики, которые могут быть использованы для построения классификационной модели для прогнозирования одобрения или отказа в выдаче кредита. Столбец loan_status служит целевой переменной для задачи классификации.
